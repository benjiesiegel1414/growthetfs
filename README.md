# GrowthETFs.com

[![GitHub Pages](https://github.com/benjiesiegel1414/growthetfs/actions/workflows/pages.yml?label=GitHub%20Pages)](https://growthetfs.com) <!-- Optional badge once you add Actions -->

In-depth research, comparisons, performance data, and insights on high-growth ETFs for long-term investors.

Live site: https://growthetfs.com

## About
This repository powers **GrowthETFs.com** — a resource for analyzing growth-oriented exchange-traded funds (e.g., VUG, QQQ, SCHG, ARKK alternatives, sector-specific growth plays). Content includes ETF profiles, historical returns, expense ratios, holdings breakdowns, and strategy guides.

Built with static files (HTML/CSS/JS) and hosted on GitHub Pages.

## Tech Stack (Planned/Current)
- HTML5 + Tailwind CSS (via CDN for now)
- Potential: Astro / Hugo / Eleventy for better content management
- Client-side charts: Chart.js or Lightweight Charts
- Data: Static JSON or client-side fetches from free APIs (Yahoo Finance, Alpha Vantage)

## Getting Started (Local Development)
1. Clone the repo: `git clone https://github.com/benjiesiegel1414/growthetfs.git`
2. Open `index.html` in a browser or use Live Server (VS Code extension)
3. Make changes → Commit & push → Auto-deploys to GitHub Pages
